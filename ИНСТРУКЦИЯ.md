# 🚀 Инструкция по деплою ТеорМех Бота

## Шаг 1 — Захостить Mini App на Netlify (бесплатно, 2 минуты)

1. Открой https://app.netlify.com/drop
2. Перетащи файл **index.html** прямо на страницу
3. Netlify сразу даст ссылку вида: https://amazing-name-123.netlify.app
4. Скопируй эту ссылку — это твой MINI_APP_URL

> Необязательно: зарегистрируйся на netlify.com чтобы получить постоянную ссылку
> и возможность обновлять файл. Можешь переименовать сайт в настройках.

---

## Шаг 2 — Вставить URL в bot.py

Открой файл bot.py и замени строку:
```
MINI_APP_URL = "ВСТАВЬ_СЮДА_СВОЙ_NETLIFY_URL"
```
на:
```
MINI_APP_URL = "https://твой-сайт.netlify.app"
```

---

## Шаг 3 — Запустить бота

### Вариант A: Локально на компьютере/Mac
```bash
pip install pyTelegramBotAPI
python bot.py
```
Бот работает пока открыт терминал.

### Вариант B: Railway (бесплатный хостинг для бота, рекомендую)
1. Зарегистрируйся на https://railway.app
2. Нажми "New Project" → "Deploy from GitHub"
   (или "Empty Project" → загрузи файлы)
3. Добавь файлы: bot.py, requirements.txt
4. Railway сам установит зависимости и запустит бота 24/7

### Вариант C: PythonAnywhere (бесплатно)
1. Зарегистрируйся на https://pythonanywhere.com
2. Открой консоль Bash
3. Загрузи bot.py и requirements.txt
4. pip install pyTelegramBotAPI
5. python bot.py &

---

## Шаг 4 — Настроить @BotFather (опционально но красиво)

Напиши @BotFather:
- /setdescription — описание бота
- /setabouttext — текст "О боте"
- /setuserpic — аватарка
- /setcommands — меню команд:
  ```
  start - Главное меню
  app - Открыть приложение
  help - Помощь
  ```
